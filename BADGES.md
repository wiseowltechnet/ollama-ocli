## Installation Section Update

```markdown
## Installation

### Via Cargo (Recommended)
```bash
cargo install ocli
```

### Via Homebrew
```bash
brew install wiseowltechnet/ocli/ocli
```

### From Source
```bash
git clone https://github.com/wiseowltechnet/ollama-ocli
cd ollama-ocli
cargo build --release
```

## Quick Start

```bash
# Start OCLI
ocli

# Show help
/help

# Create a plan
/plan Build a REST API

# Read a file
/read src/main.rs

# Monitor stats
/monitor
```
```
