# OCLI v0.3.0 Social Media Announcements

## Reddit - r/rust

**Title**: [Release] OCLI v0.3.0 - Enterprise AI Coding Assistant with LCARS Styling

**Body**:
I just released OCLI v0.3.0, an enterprise-grade AI coding assistant for Ollama with a Star Trek LCARS-inspired interface!

**Key Features:**
- Command history with navigation and tab completion
- Syntax highlighting with LCARS theme
- Smart context management with auto-pruning
- AI-powered suggestions and typo correction
- Built-in diff viewer with side-by-side comparison
- Multi-model support with easy switching
- Performance metrics tracking
- MCP integration

**Tech Stack:**
- rustyline, syntect, indicatif
- 57 tests across 5 frameworks

**Install:**
```bash
cargo install ocli
```

**Links:**
- GitHub: https://github.com/wiseowltechnet/ollama-ocli
- Crates.io: https://crates.io/crates/ocli
